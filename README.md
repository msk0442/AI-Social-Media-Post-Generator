Transform your ideas into professional, high-engagement social media content in seconds. Powered by **Google Gemini 2.5 Flash** and **Imagen 3.0**, this generator creates breathtaking conceptual imagery and compelling copy tailored for your favorite platforms.

---

## 🚀 Key Features

- **🌈 Premium UI/UX**: Built with a sleek, modern glassmorphic theme and high-end animations using **Framer Motion**.
- **🧠 Intelligent Content**: Uses state-of-the-art LLMs to generate professional-grade social media copy.
- **🖼️ Conceptual Imagery**: AI-generated images that are abstract, minimalist, and perfectly aligned with your topic.
- **📱 Platform Optimized**: Support for **LinkedIn**, **Instagram**, **Facebook**, **Twitter (X)**, and **Stories**, with correct aspect ratios automatically applied.
- **🎛️ Fully Customizable**: Edit the generated text instantly or refresh for new variations.
- **🛠️ Open Source**: Download, use your own API keys, and deploy anywhere.

## 🏁 Getting Started

Follow these simple steps to run the generator on your local machine:

### 1. Prerequisite: Get your Gemini API Key
- Go to [Google AI Studio](https://aistudio.google.com/) and create a free API Key.

### 2. Installation
```bash
git clone https://github.com/msk0442/AI-Social-Media-Post-Generator.git
cd AI-Social-Media-Post-Generator
npm install
```

### 3. Environment Setup
Create a `.env` file in the root directory:
```bash
cp .env.example .env
```
Add your Gemini API Key to the `.env` file:
```env
VITE_GEMINI_API_KEY=your_gemini_api_key_here
```

### 4. Launch
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) and start generating!
